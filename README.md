# Handbag Store Ecommerce Application

A modern ecommerce platform for handbags and accessories built with React, Node.js, and Firebase.

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express
- **Database**: Firebase Firestore
- **Image Storage**: Cloudinary
- **Deployment**: Vercel (Frontend), Render (Backend)

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Firebase account
- Cloudinary account
- Git
